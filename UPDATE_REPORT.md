## ✨ 更新内容

### 优化新闻搜索输出格式

**用户需求**: 当使用 `/brief <关键词>` 搜索新闻时，先汇总新闻内容，末尾再附链接。

**更新前**:
```
1. 新闻标题
   摘要内容
   [链接](url)  ← 每条新闻都带链接

2. 新闻标题
   摘要内容
   [链接](url)
```

**更新后**:
```
## 📰 新闻内容汇总

1. **新闻标题**
   来源：BBC | 时间：2 小时前
   摘要内容...

2. **新闻标题**
   来源：AP | 时间：3 小时前
   摘要内容...

---

## 🔗 完整新闻链接

1. **新闻标题**
   来源：BBC
   链接：url

2. **新闻标题**
   来源：AP
   链接：url
```

## 📝 修改的文件

### 1. `src/summarize.js`
**修改内容**:
- 优化 `formatBriefing()` 函数
- 将链接统一移到末尾
- 添加关键词显示
- 改进排版格式

**代码变更**: +30 行

### 2. `src/format-search.js` (新增)
**功能**:
- 专门用于搜索结果的格式化
- 内容汇总区（无链接）
- 链接汇总区（末尾）
- 时间格式化（刚刚/X 小时前/昨天）

**代码行数**: 65 行

### 3. `commands/brief.js`
**修改内容**:
- 引入 `format-search.js`
- 重写 `handleSearch()` 函数
- 使用新的格式化函数
- 改进错误处理

**代码变更**: +20 行

## 🎯 功能特性

### 搜索简报格式

**头部信息**:
- 🔍 搜索关键词
- 📊 相关新闻数量
- ⏰ 搜索时间

**内容汇总区**:
- 新闻标题（加粗）
- 来源 + 时间
- 摘要内容（150 字）
- 不显示链接（保持清爽）

**链接汇总区**:
- 完整标题
- 来源媒体
- 原始链接
- 方便点击访问
