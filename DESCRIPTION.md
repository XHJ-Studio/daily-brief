# Daily Brief (小黄鸡每日简报)

🤖 AI-driven personalized news briefing generator, like President's Daily Brief (PDB)

## Features

- 📡 145+ RSS feeds from 65+ global media outlets
- 🌍 Global coverage: US/China/Europe/Asia
- 🤖 AI-powered summaries
- ⏰ Scheduled daily delivery at 09:00
- 🎯 Smart filtering based on interests
- 📊 Quality scoring
- 🔄 Auto-deduplication
- 📱 Multi-platform: Discord/Telegram/WeChat

## Quick Start

```bash
git clone https://github.com/xhjgfstudio/daily-brief.git
cd daily-brief
npm install
npm start
```
