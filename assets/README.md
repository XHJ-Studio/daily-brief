# Assets

此目录存放项目资源文件：

- `community-qr.png` - 社群二维码（用于分享，非联系方式）
- `icon.png` - 项目图标
- `screenshot1.png` - 截图 1
- `screenshot2.png` - 截图 2

---

## 使用说明

### 社群二维码

`community-qr.png` 用于 README 和 SKILL.md 中的社群分享，**不作为联系方式公开**。

### 添加图片

1. 将图片文件放入此目录
2. 在 README/SKILL.md 中引用：`![描述](./assets/图片名.png)`
3. 提交更改
